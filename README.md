# Encryption Decryption Tool Using Java GUI

A secure Java-based desktop application that allows users to encrypt and decrypt text and files using password-based AES-GCM encryption.

The project demonstrates practical implementation of:
- Java Swing GUI development
- Object-Oriented Programming
- Java Cryptography Architecture
- AES-GCM authenticated encryption
- PBKDF2 password-based key derivation
- Secure file handling
- Exception handling and validation

Built for educational purposes to understand how modern encryption systems protect sensitive information.
